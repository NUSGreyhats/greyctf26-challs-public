## Design Notes

- The public checker is binary-only.
- The public checkpoint is a compact normal `state_dict` under 2 MB. `print(model)` shows an `Embedding`, recurrent core, linear readout, and classifier head rather than an empty tensor wrapper.
- Large per-step cell transports are regenerated by the public runtime from a seed instead of stored in `model.pt`.
- The public runtime exposes only checker-facing execution. Trace helpers and named cell/terminal debug APIs were removed; the private solver instruments private attributes locally.
- The private generator still compiles semantic verifier modules into hard recurrent gates, but the public checkpoint stores dense module-style tensors rather than sparse index/value tensors, integer routing maps, old symbolic tables, or dense projection surfaces.
- Public tensors do not include char masks, pair tables, triple tables, target residues, causal-parent metadata, or semantic verifier key names.
- The explicit phase register was removed from the recurrent state; the loop index supplies phase while previous-character registers, real/decoy latches, and terminal memory constraints preserve the mechanistic solve path.
- The official solver uses activation/chart recovery, previous-character register recovery, causal patching, recovered transition relations, and Z3 uniqueness proof.
